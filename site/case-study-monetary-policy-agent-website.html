Monetary Policy Agent Website Use Case

Challenge

As the project grew, it needed a clear website-level narrative that helps different audiences find the right artifacts quickly: architecture for engineers, run evidence for operators, and summaries for stakeholders.

Solution

This use case frames the Monetary Policy Agent as a website experience with linked entry points for system design, analytics outputs, execution evidence, and governance documentation.

Every major page is linked below so the site can be used as a project control panel during demos, reviews, and handoffs.
