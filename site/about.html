About This Project

FOMC Agent Lab is a workshop-style demo that implements a complete pipeline using the requirements in docs/instructions.md.

  • Ingest: BLS LABSTAT flat files + DataUSA population API into S3 raw buckets.
  • Process: parse/update + enrich steps are separated from ingestion.
  • CDC: S3 notifications → SQS → Lambda for durable, decoupled processing.
  • Publish: static website + JSON artifacts for dashboards and charts.

Interview Talking Points

Why “raw-first” ingestion?

Raw buckets preserve the original data and make every downstream step reproducible and re-runnable.

Why queue-based CDC?

SQS provides backpressure, retries, and a DLQ — turning “events” into reliable work.

How do you test this?

Unit tests mock AWS with moto; LocalStack simulates end-to-end behavior locally.

What would you change for production?

Use schema versioning, data contracts, governance, and a stronger “silver/gold” layout.
